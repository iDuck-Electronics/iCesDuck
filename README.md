# iCesDuck
## Placa FPGA para Raspberry Pi 4B

iCesDuck es una placa de control basada en FPGA, diseñada para desarrollar y depurar lógica digital directamente sobre una Raspberry Pi 4B. Integra una FPGA, conversores DAC/ADC y un convertidor lógico para adaptación de niveles, permitiendo probar código Verilog y VHDL en hardware real sin montar tanto circuito externo.

### Características principales

- Compatíble con Raspberry Pi 4B como “cerebro” y entorno de desarrollo  
- FPGA para implementación y prueba de diseños en Verilog y VHDL  
- Integrados DAC y ADC para señales analógicas de entrada/salida  
- Convertidor lógico para distintos niveles de voltaje en E/S  
- Conector flex hacia placa de expansión para llevar señales a pines, bornes o módulos externos  

### Uso típico

- Laboratorio de prácticas de sistemas digitales y mixtos  
- Prototipado rápido de módulos lógicos y de control  
- Experimentación con interfaces analógico–digitales usando la Raspberry como host
